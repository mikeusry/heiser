# Heiser Group - Claude Code Context

Professional cleaning services website for The Heiser Group (Lake County, IL since 1978).

---

## Quick Reference

| Item | Value |
|------|-------|
| **Tech Stack** | Astro 5 + Tailwind CSS 4 + TypeScript |
| **Hosting** | Cloudflare Pages (manual deploy via Wrangler) |
| **Dev Server** | `npm run dev` → http://localhost:4321 |
| **Image CDN** | Cloudinary (southland-organics cloud) |
| **Git Remote** | https://github.com/mikeusry/heiser.git |

```bash
npx wrangler pages deploy dist --project-name=heiser
```
