# Source

Astro site source code for The Heiser Group website.

## Directory Structure

```
src/
├── components/     # Reusable Astro components
├── content/        # JSON content (pages, images)
├── layouts/        # Page layouts
├── lib/            # TypeScript data modules
├── pages/          # Astro pages (routes)
└── styles/         # CSS files
```

## Tech Stack

- **Framework:** Astro 5
- **Styling:** Tailwind CSS 4
- **Images:** Cloudinary CDN
- **Hosting:** Cloudflare Pages
