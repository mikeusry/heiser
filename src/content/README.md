# Content

Static content data.

## Files

| File | Description |
|------|-------------|
| `pages-index.json` | Index of all pages |
| `images.json` | Original image URLs from crawl |
| `cloudinary-images.json` | Uploaded Cloudinary URLs |

## `pages/` Directory

Individual page content JSON files extracted from Apify crawl.

Each file contains:
- `slug` - URL slug
- `title` - Page title
- `description` - Meta description
- `markdown` - Page content as markdown
- `text` - Plain text content

## Generation

Content generated by `scripts/parse-crawl-data.js` from `heiser-site.json`.
