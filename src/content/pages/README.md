# Pages Content

JSON content files for each page, extracted from Apify crawl.

## Structure

Each `.json` file contains:
```json
{
  "slug": "page-name",
  "url": "https://...",
  "title": "Page Title",
  "description": "Meta description",
  "markdown": "Page content...",
  "text": "Plain text...",
  "jsonLd": [],
  "screenshotUrl": ""
}
```

Generated by `scripts/parse-crawl-data.js`.
